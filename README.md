# Evos Fast Food Telegram Bot

This bot is designed for **Evos Fast Food** and offers the following features:

## Features:
- View and order from the menu.
- Manage user information.
- Track order status.
- Multi-language support.

## Technologies:
- **Python**: For backend and bot logic.
- **Aiogram**: For interacting with Telegram API.
- **JSON**: For data storage.
